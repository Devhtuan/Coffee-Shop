							        Hướng dẫn để cài đặt và build lên chương trình.

Bước 1 : Cài đặt XAMPP mới nhất, cài đặ Visual Studio Code
Bước 2 : Khởi động XAMPP -> vào trang PhpMyAdmin
Bước 3 : Mở source code bằng Visual Studio Code, chạy Terminal nhập lệnh: 
	 + $ php artisan --version để kiểm tra phiên bản Laravel Framework
	 + $ php artisan serve và vào link http://127.0.0.1:8000/ hiện ra để chạy chương trình web bán cafe
Bước 4 : thay đổi link http://127.0.0.1:8000/admin để vào trang quản trị admin
